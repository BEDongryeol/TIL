## Java 1.8의 Functional Interface

Functional Interface는 함수형 인터페이스로 1개 이상의 추상 메서드를 가지고 있는 인터페이스이다.

Single Abstract Method라고 불리기도 한다.

### 사용하는 이유

Java의 람다식은 함수형 인터페이스로만 접근이 되기 때문에 사용한다.

### 예시

- Runnable -> run()

- Consumer -> accept()

- Predicate -> test()

- Supplier -> get()

- Callable -> submit()
